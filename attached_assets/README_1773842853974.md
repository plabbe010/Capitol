# 🏛️ Capitol Watch

Track, follow, and analyze congressional stock disclosures — with AI-powered signals and unusual trade detection.

---

## Replit Setup (5 minutes)

### 1. Create a new Replit
- Go to replit.com → **Create Repl**
- Choose **"Import from GitHub"** or **"Node.js"** template
- Upload these files, or paste them in

### 2. Add your API key as a Secret
- In Replit, click **Tools → Secrets**
- Add a new secret:
  - Key: `QUIVER_API_KEY`
  - Value: your key from [quiverquant.com](https://quiverquant.com)
- **Never put your key in code or .env files in Replit**

### 3. Install dependencies & run
In the Replit Shell:
```bash
npm run install:all
npm run build
npm start
```

Or just click **Run** — Replit will use `.replit` to handle everything.

### 4. Enable Always On
- Go to your Repl's settings
- Enable **Always On** so it stays live 24/7

## How the data flows

```
Browser → /api/trades → Express server → QuiverQuant API
                              ↑
                    QUIVER_API_KEY (Replit Secret)
                    15-min cache (NodeCache)
```

- Your API key **never touches the browser**
- Responses are cached for 15 minutes to avoid hitting rate limits
- Hit **↻ Refresh** in the UI to bust the cache and force a fresh pull

## Features

| Feature | Details |
|---|---|
| Live data | House + Senate via QuiverQuant free tier |
| Flag detection | First buys in years, committee conflicts, large obscure positions |
| AI signals | Buy/Hold/Sell + confidence % + plain-English explanation |
| AI summary | Weekly digest of congressional trading patterns |
| Watchlist | Star tickers and members, get alert banners on new trades |
| Demo mode | Falls back to demo data if no API key / server unreachable |

## Local Development

```bash
# Install everything
npm run install:all

# Copy env template and fill in your key
cp .env.example .env

# Run both server and client with hot reload
npm run dev
```

Frontend runs on `http://localhost:5173`  
Backend runs on `http://localhost:3001`

## Getting a QuiverQuant API Key (free)

1. Go to [quiverquant.com](https://quiverquant.com)
2. Create a free account
3. Go to your profile → API key
4. Copy and paste into Replit Secrets as `QUIVER_API_KEY`

The free tier covers congressional trading data for both chambers.
